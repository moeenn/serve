## Static file server
A simple static file server.


#### Usage
```bash
Usage of serve:
  -dir string
        Directory to serve. (default ".")
  -port int
        Port to run file server on. (default 3000)
```

#### Installation
```bash
# install directly from git.
$ go install github.com/moeenn/serve@v1.0.0

# manually build and install.
$ make install
```
